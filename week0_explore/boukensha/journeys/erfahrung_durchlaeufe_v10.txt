# Kern-Erkenntnisse aus diesem Lauf:
# - Der VOID tritt auf, wenn Navigation zu lange fehlschlaegt oder Agent zu lange inaktiv ist.
# - Void ist ein Zuruecksetzen in einen Startzustand: HP/MV erhalten, aber Inventar kann verloren gehen.
# - API-Timeouts treten auf, wenn zu viele Anfragen hintereinander ohne Output erfolgen.
# - Die fest etablierte Route Common Square -> west -> Eastern End of Poor Alley ist sicher und reproduzierbar.
# - Grubby Inn (sued von Eastern End of Poor Alley) ist ein bekannter, sicherer Ankerpunkt.
# - Armory, Grunting Boar Inn, Guild of Swordsmen sind zusaetzliche Navigations-Anker.
# - Practice Yard ist dort, wo man Fertigkeiten trainieren kann (nach Level-Up).
# - Bank/ATM ermoeglicht balance, withdraw, deposit.
# - Die Main Street ist ein hohes Wachen-Durchgangsgebiet und sollte minimiert werden.
# - Zu viele aufeinanderfolgende Moves fuehren zu API-Timeouts.
# - Ein Move pro Befehl ist sicher; zwei Moves hintereinander erhoehen Timeout-Risiko.

# Neue Kern-Erkenntnis: VOID-ZUSTAND
# - Der Void tritt auf nach ungueltige Navigationen oder langen Wartepausen.
# - Im Void: Raumname/Exits fehlen, nur "You are hungry/thirsty" bleibt.
# - Ausweg aus dem Void: score, look, dann einen sicheren Move ausfuehren (z.B. north).
# - Nach Void muss die Position mit look neu bestimmt werden.
# - Void-Trigger vermeiden: nicht zu schnell zu viele Moves, warten auf Output.

# Neue Kern-Erkenntnis: BANK / ATM
# - Bank ist verfuegbar in bekannten Raeumen (wahrscheinlich Temple Square / Market Square).
# - Befehle: balance (saldo pruefen), withdraw <summe> (Geld abheben), deposit <summe> (Geld einzahlen).
# - Nur nutzen, wenn sicher und Geldmanaagement noetig ist (spaeter, nach stabilem Farm).

# Neue Kern-Erkenntnis: NEUE NAVIGATIONSPUNKTE
# - Grubby Inn: sued von Eastern End of Poor Alley, sichere Alternative.
# - Guild of Swordsmen: sued von Main Street, enthält Practice Yard / Trainingsraum.
# - Armory: sùdlich von Main Street, eventuell Ausruestungsquelle (spaeter relevant).
# - Grunting Boar Inn: oestlich vom Market Square, eventuell Ressource.
# - Practice Yard: suedlich vom Bar of Swordsmen, fuer Skill-Training nach Level-Up.

# Neue Kern-Erkenntnis: API-TIMEOUTS
# - API-Timeouts treten auf bei zu schnellen aufeinanderfolgenden Anfragen ohne Output.
# - Loeschen: nach jedem Move warten bis Output kommt, bevor nächster Move.
# - Batch-Planung sollte "kurze Pausen" enthalten, nicht nur Kommandos hintereinander.
