# Ziele der Woche 1

Ich möchte einen grundlegenden Agent erzeugen, der alle Komponenten und Funktionen beinhaltet.

Ich denke folgendes sollte vorhanden sein:
- eine einfache Agenten Schleifen
- Tool Registry mit Tools (arbeiten können)
- Er sollte mit verschiedenen Backends arbeiten können
- er soll Logs erzeugen
- er sollte eine DSL haben und das SDK verwenden
- er soll ein globales Binary haben, dass mit der CLI arbeitet
- er soll optional in CLI-Mode arbeiten können
- er soll den Kontext steuern und Messages zusammenfassen, wenn diese zu lang sind
- er soll ein config-Verzeichnis haben

Weitere Funktionen könnte er noch heben:
- log Visualisierung im Browser
